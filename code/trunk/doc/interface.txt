# 与统一登录账号系统的接口需求
* 1. 验证账号和密码是否符合格式.
* 2. 判断符合格式的账号和密码是否登录成功.
* 3. 密码传输过程的加密方式.
* 4. 账号登录成功,修改用户的最后一次登录时间.
* 5. 账号登录成功,获取用户的详细信息.
* 6. 账号登录成功,获取用户下的所有公司的详细信息.
* 7. 账号登录成功,确定以用户的什么属性或什么机制作为系统间交互的唯一凭证.
* 8. 用户注册接口
* 9. 用户忘记密码接口
* 10.用户修改密码接口.
* 11.登录用户是代理商还是普通浏览用户.
